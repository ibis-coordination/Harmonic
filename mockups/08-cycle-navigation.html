Option C: Segmented Tabs

Horizontal tabs showing recent cycles for quick switching

Pros

Pros

  • Best of both worlds

  • Flexible navigation

Cons

  • More complex interaction
  • Two ways to do same thing

Implementation Notes

Technical considerations for cycle navigation

URL Structure

Navigation can use the existing /cycles/:name route pattern:

  • /studios/handle/pulse - Current cycle (based on tempo)
  • /studios/handle/pulse?cycle=last-week - Last week
  • /studios/handle/pulse?cycle=yesterday - Yesterday
  • /studios/handle/pulse?cycle=this-month - This month

Available Cycle Names

From Cycle#cycle_options:

  • Days: today, yesterday, tomorrow
  • Weeks: this-week, last-week, next-week
  • Months: this-month, last-month, next-month
  • Years: this-year, last-year, next-year

Cycle Navigation Logic

Use Cycle#previous_cycle method for backward navigation:

  • today → yesterday
  • this-week → last-week
  • this-month → last-month

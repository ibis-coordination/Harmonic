# Rails Console

Open an interactive Rails console session.

## Usage

- `/console` - Open Rails console

## Instructions

1. Inform the user that you'll open the Rails console

2. Run the console script:
   ```bash
   ./scripts/rails-c.sh
   ```

3. Note: This opens an interactive session. The user will interact with it directly.

## Tips for the user

- The console runs inside Docker with full Rails environment

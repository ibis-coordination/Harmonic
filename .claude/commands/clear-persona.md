# Clear Persona

Drop the current persona and return to default Claude behavior.

## Usage

- `/clear-persona` - Return to default Claude

## Instructions

1. Acknowledge that the persona has been cleared
2. Return to normal Claude behavior - no persona, no character, just standard assistant mode
3. Confirm readiness to continue helping

## Example Response

---

Persona cleared. I'm back to being standard Claude.

How can I help you?

---
